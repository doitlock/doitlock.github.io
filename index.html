[HTML / CSS / PHOTOSHOP]

1차 포트폴리오의 메인화면!
시멘틱 태그 header, aside, main, footer를 사용하여 레이아웃을 잡았다.

header, aside, footer는 position을 fixed로 설정해 고정을 시켰다.
중앙의 main부분만 스크롤이 된다!

클론 작업 사이트의 썸네일박스는 display:flex 를 사용하여 정렬했다.
flex를 처음 써봤는데 float보다 훨씬 쉽게 정렬되었다.
썸네일에 마우스를 올리면 썸네일위로 투명한 배경에 텍스트가 보이도록 했다.

수업중에 배웠던 패럴랙스를 사용하여 스크롤해도 배경이미지가 고정되어 보이도록 했다.
이미지위에 넣은 텍스트는 '목표는 취업이다' 이다.

버튼은 a태그를 사용하여 만들었다.
마우스를 올리면 백그라운드 색상이 1초동안 바뀌도록 transition을 사용했다.
아이콘은 font awesome에서 가져왔다.

이전에 그려둔 그림들을 사용해보면 재미있을 것 같아서 넣어보았다.
rotate, infinite효과로 저 그림은 3초동안 한 바퀴씩 영원히 돌아갈 것 이다.

footer부분은 fixed를 사용하여 밑에 고정시켜 두었기에,
main화면의 하단에 따로 footer부분을 만들어두었다.

원페이지로 제작된 라인프렌즈 홈페이지. 스크립트를 처음으로 사용해 보았다!
[HTML / CSS / JAVASCRIPT]

라인프렌즈의 인트로 메인 화면!
시멘틱 태그 header, section, footer를 사용하여 레이아웃을 잡았다.
(롤배너는 플러그인 slick slider 사용)

로고는 h1태그에 background-image를 사용하여 이미지를 넣었고,
h1의 텍스트는 보이지 않도록 font-size:0 을 주었다.

메인 메뉴는 float을 사용했다. 부모 태그에는 overflow:hidden이 아닌
:after선택자를 만들어서 content를 주었다!
(after content가 아니라 부모태그에 overflow:hidden를 넣어 대응하면 때때로 컨텐츠가 전부 출력되지 않는다고 하니 :after content사용을 습관화 해야겠다!)

스크롤을 내렸을 때 헤더가 고정되도록 position:fixed! 롤배너 위에서는 보이지 않던 헤더의 배경색상이 흰색으로 바뀌도록 스크립트의 스크롤이벤트를 사용했다. 서로 비슷한 6개 section의 가로 사이즈와 위아래 여백에 max-width: 1020px, margin: 0 auto, padding: 110px 0 120px을 공통으로 주었다.

dl, dt태그와 float을 사용하여 컨텐츠를 정렬했다.
컨텐츠의 width값을 똑같이 3등분으로 나누기 위해 width: calc(100%/3)을 사용해보았다!

ul, li태그를 사용하여 정렬했고, li안의 a태그에 컨텐츠 내용을 넣었다.
이미지와 텍스트의 widht는 각각 65%, 35%로 나누었다. 밑의 pagenavigation부분도 ul, li태그를 사용했고 li를 옆으로 정렬하기 위해 display:inline-block을 주었다.

역시 ul, li태그 사용! 각 컨텐츠마다 width를 50%! 모바일 반응형에서는 width를 100%!

table 태그를 사용하여 공지사항 게시판 부분을 만들었다.
thead, tbody로 영역을 나누었고, thead에는 tr과 th를 tbody는 tr과 td를 사용하여 텍스트 입력!
표의 열을 묶기 위해 colgroup태그를 사용했다.

footer부분은 내용 전체를 footer_wrap으로 묶어 position:relative로!
로고, 푸터 컨텐츠와 sns아이콘은 position:absolute로 위치를 잡아주었다.

현재 반응형으로 작업중이다!
